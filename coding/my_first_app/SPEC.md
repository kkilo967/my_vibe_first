# KKKILO 个人网站规范文档

## 1. 项目概述

- **项目名称**: KKKILO 个人网站
- **项目类型**: 单页个人网站
- **核心功能**: 展示个人简介、作品集、联系方式
- **目标用户**: 潜在雇主、客户、合作伙伴

## 2. UI/UX 规范

### 2.1 布局结构

**页面区块**:
- **导航栏**: 固定顶部，包含logo和导航链接
- **Hero区域**: 全屏高度，首屏展示个人品牌
- **关于我**: 个人介绍区域
- **作品集**: 项目展示网格
- **联系我**: 联系表单/信息
- **页脚**: 版权信息

**响应式断点**:
- 移动端: < 768px
- 平板: 768px - 1024px
- 桌面: > 1024px

### 2.2 视觉设计

**色彩系统**:
- 主色(薄荷蓝): #5AC8C8
- 主色深色: #4BA8A8
- 主色浅色: #A8E6E6
- 背景色: #FAFCFC
- 深色背景: #F0F7F7
- 文字主色: #1A1A2E
- 文字次色: #6B7280
- 白色: #FFFFFF

**字体系统**:
- 标题字体: "SF Pro Display", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
- 正文字体: "SF Pro Text", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
- Hero标题: 72px (桌面) / 40px (移动)
- 区块标题: 48px (桌面) / 32px (移动)
- 副标题: 24px
- 正文: 18px
- 小字: 14px

**间距系统**:
- 区块间距: 120px (桌面) / 80px (移动)
- 内容区最大宽度: 1200px
- 卡片间距: 32px
- 元素间距: 16px / 24px

**视觉效果**:
- 卡片阴影: 0 4px 24px rgba(90, 200, 200, 0.08)
- 悬停阴影: 0 8px 40px rgba(90, 200, 200, 0.15)
- 圆角: 16px (卡片) / 24px (大卡片) / 9999px (按钮)
- 背景: 微妙的薄荷蓝渐变和几何图案

### 2.3 组件设计

**导航栏**:
- 高度: 72px
- 背景: 毛玻璃效果 (backdrop-filter: blur(20px))
- 悬停: 文字颜色过渡到主色
- 滚动后: 背景透明度增加

**Hero区域**:
- 全屏高度 (100vh)
- 动态背景: 流动的几何图形/波浪
- 标题: 渐入动画
- 副标题: 打字机效果
- CTA按钮: 悬停涟漪效果

**关于我卡片**:
- 大头像: 200px圆形
- 技能标签: 圆角胶囊样式
- 描述文字: 两端对齐

**作品集卡片**:
- 3列网格 (桌面) / 2列 (平板) / 1列 (移动)
- 悬停: 轻微上浮 + 阴影加深
- 图片比例: 16:10
- 标签: 分类显示

**联系表单**:
- 输入框: 底部边框样式
- 提交按钮: 填充主色
- 验证状态: 成功/错误提示

### 2.4 动画效果

**入场动画**:
- 元素从下方淡入上移 (translateY + opacity)
- 交错延迟: 100ms递增
- 时长: 0.8s
- 缓动: cubic-bezier(0.16, 1, 0.3, 1)

**流动背景动画**:
- SVG波浪/几何图形持续流动
- 时长: 20s循环
- 方向: 水平流动

**交互反馈**:
- 按钮悬停: scale(1.02) + 阴影
- 卡片悬停: translateY(-8px)
- 链接悬停: 颜色过渡 0.3s
- 滚动触发动画: IntersectionObserver
